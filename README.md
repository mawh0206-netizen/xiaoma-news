# 小马看世界

个人每日新闻工作台。每天北京时间 09:00 前发布中文晨报，聚焦 AI、科技、企业商业、财经、房地产、国内外汽车产业与国际要闻。

## 本地预览

在项目目录运行静态服务器，然后访问终端显示的地址：

```powershell
python -m http.server 4173
```

## 内容原则

- 国际核心来源：Reuters、BBC、Financial Times、The Guardian、TechCrunch。
- 国内核心来源：第一财经、财联社、证券时报、36氪、澎湃新闻、界面新闻、经济观察报，以及盖世汽车、中国汽车报、中国汽车流通协会等垂直媒体与行业机构。
- 仅展示中文标题、短摘要、AI 决策提示、来源和原文链接，不复制媒体全文。
- 首页约 20 条必读，栏目页保留更多候选内容。
- 栏目配额：AI/科技/企业商业/财经至少 8 条，房地产与国际要闻至少 6 条，汽车产业至少 10 条；首页仍保持约 20 条，控制晨读负担。
- 每日任务建议 08:15 启动，09:00 前完成发布。
- 抓取或发布失败时保留上一版，并通过本机现有 QQ SMTP 配置发送异常邮件。

`data/news.json` 是网站唯一的内容入口。自动任务生成新内容后覆盖该文件并触发静态部署。
